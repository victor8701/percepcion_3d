# Percepción 3D

Repositorio para las prácticas de la asignatura de Percepción 3D, desarrolladas utilizando MATLAB y enfocado en el procesamiento y análisis de datos tridimensionales.

## Estructura del repositorio

* **[Practica 1](Practica%201/)**: Introducción práctica al análisis, preprocesamiento y segmentación de nubes de puntos (Point Clouds).
  * 🗃️ Resolución algorítmica y documentación de los 9 pasos del guion base.
  * 🚀 **Proyectos Finales:** Integra la canalización evaluable del Aislamiento Algorítmico del Teléfono y del Inventario Urbano y Conteo de Entidades.
* **[Practica 2](Practica%202/)**: Reconstrucción 3D de escenas a partir de nubes de puntos mediante registro automático y fusión incremental.
  * 📐 Transformaciones geométricas 3D (traslación, rotación) y registro ICP con `pcregistericp`.
  * 🏠 **Reconstrucción RGB-D:** Fusión incremental de una secuencia de capturas Kinect acumulando transformaciones para reconstruir una sala completa.
